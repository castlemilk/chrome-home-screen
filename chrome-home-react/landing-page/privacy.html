Privacy Policy

Last updated: January 7, 2024

Our Commitment to Privacy

Chrome Home is committed to protecting your privacy. This privacy policy explains what information we collect (spoiler: almost none), how we use it, and your rights regarding your data.

Information We DO NOT Collect

Chrome Home is designed with privacy as a core principle. We DO NOT:

  • Track your browsing activity
  • Collect personal information
  • Store data on external servers
  • Use analytics or tracking cookies
  • Require user accounts or registration
  • Share any data with third parties
  • Display advertisements

Data Storage

All data is stored locally on your device using Chrome's storage API:

  • Settings: Your preferences (background choice, widget configuration) are stored locally
  • Widget Data: Todo items, selected stocks, weather locations are stored locally
  • Cache: API responses are cached locally to improve performance

This data never leaves your device and is not accessible to us or any third party.

API Connections

Chrome Home connects to the following services only when specific widgets are used:

  • Weather Data: wttr.in (only when weather widget is active)
  • Stock Data: Yahoo Finance API (only when stock widget is active)
  • Location Search: Open-Meteo Geocoding (only during location search)

These connections are made directly from your browser. We do not proxy or store these requests.

Permissions

Chrome Home requires minimal permissions:

  • Storage: To save your settings and widget data locally
  • Host Permissions: To fetch weather and stock data when widgets are used

Data Security

Your data security is ensured through:

  • All data stored locally on your device
  • No external server communication for user data
  • Open source code for complete transparency
  • Regular security updates

Your Rights

You have complete control over your data:

  • Access: All your data is stored locally and accessible through the extension
  • Modification: Change any settings or data at any time
  • Deletion: Uninstalling the extension removes all associated data
  • Portability: Export your data through Chrome's sync feature

Children's Privacy

Chrome Home does not collect any information from anyone, including children under 13.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated revision date.

Open Source

Chrome Home is open source. You can review our code at any time on GitHub to verify our privacy practices.

Contact Us

If you have any questions about this privacy policy or Chrome Home's privacy practices, please contact us:

  • Email: privacy@chromehome.app
  • GitHub Issues: Report an Issue

Compliance

Chrome Home complies with:

  • Chrome Web Store Developer Program Policies
  • General Data Protection Regulation (GDPR)
  • California Consumer Privacy Act (CCPA)
